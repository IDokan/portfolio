Game Jam (3rd Indie Game Weekend)Used Engine: UnrealTeam size: 4 people teamRole: Tech-leadDuration: 48 hours

In highschool, I participated 3rd Indie Game Weekend to create game programmer's network and meet people in game programming companies.

In the game jam, my team created a game using Unreal. I used Blueprint which is Unreal's Visual Scripting.

As it was our first time working together to create a game, both my teammates and I were unfamiliar with the process of integrating individual's work into a cohesive whole.

Despite being aware of the existence of Git, the tight 48-hour timeframe did not allow me sufficient time to study and fully utilize it. Consequently, my teammates and I made the decision to work together using just one laptop for programming the game. Although it resulted in a somewhat chaotic experience, it ignited a sense of passion for collaborative work and highlighted the importance of establishing a conducive environment for teamwork.
